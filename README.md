# Dream-Stay

Dream Stay is a hotel booking website created as a group project to help users search for hotels and explore different cities easily. The website includes pages like Home, Hotels, About Us, Blogs, Sign In, and Sign Up. Users can browse hotels by cities, read travel blogs, and view offers.
